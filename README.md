### 开发/构建

``` bash
# 安装依赖
yarn

# 开发
npm run dev

# 构建
npm run build
```

### MinUI 组件安装及使用

详见 MinUI 官方文档 [结合 Wepy](https://meili.github.io/min/docs/min-cli/third-project/tutorial/with-wepy.html)
